Web 1.0

Web 1.0 refers to the first generation of the World Wide Web, which was mainly static and read-only. Websites during this era were simple and provided information without much interaction from users.

Users could only view content but could not easily comment, share, or interact with the website. Pages were typically created using basic HTML and rarely updated.

Web 1.0 is often described as the "read-only web" because users consumed information without contributing to it.
