Web 2.0

Web 2.0 refers to the second generation of the World Wide Web that focuses on user interaction, collaboration, and content sharing. Unlike Web 1.0, users can now create, edit, and share content.

It introduced interactive platforms such as social media, blogs, wikis, and video-sharing sites where users actively participate instead of only viewing content.

Examples of Web 2.0 platforms include Facebook, YouTube, Twitter (X), and Wikipedia. This era is also known as the "participatory web."
